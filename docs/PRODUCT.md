# Produto

## Princípio

O Koinonia existe para aliviar a carga de cuidado, não para criar mais trabalho.

## MVP

### Pastor

- Enxerga visão macro dos eventos da semana.
- Vê presença geral.
- Vê células em atenção.
- Vê pessoas que podem estar se afastando.

### Supervisor

- Enxerga células sob sua responsabilidade.
- Vê líderes/células em atenção.
- Acompanha presença sem microgerenciar.

### Líder

- Faz check-in simples.
- Vê quem faltou repetidamente.
- Liga, chama no WhatsApp ou marca como cuidado.

## Decisão importante

O botão principal nunca deve ser “criar tarefa”.

Os botões principais devem ser:

- Ligar
- WhatsApp
- Marcar como cuidado

A anotação é opcional.
